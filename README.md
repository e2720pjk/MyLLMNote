# MyLLMNote

This directory contains organized conversation logs and session data from Google Antigravity and Opencode, categorized by project.

## Directory Structure

```
MyLLMNote/
├── CodeWiki/
│   ├── Antigravity/      # Antigravity conversation logs related to CodeWiki
│   └── Opencode/         # Opencode session data files for CodeWiki projects
└── llxprt-code/
    ├── Antigravity/      # Antigravity conversation logs related to llxprt-code
    └── Opencode/         # Opencode session data files for llxprt-code projects
```

## Contents

### CodeWiki
Includes logs for projects identified as `CodeWiki` or `CodeWiki-related`.
- **Antigravity**: Conversations involving CodeWiki features, documentation, and logic.
- **Opencode**: User activity sessions from CodeWiki worktrees.

### llxprt-code
Includes logs for projects identified as `llxprt-code` (including AST tools, etc).
- **Antigravity**: Conversations focusing on AST manipulation, code editing tools, and `llxprt` specific development.
- **Opencode**: User activity sessions from llxprt-code worktrees.
