# 簡化的執行流程

OpenCode 探索任務系統透過 OpenCode 來處理所有執行邏輯。

---

## Cron 任務設置

使用 OpenClaw cron 系統每天執行一次：

```bash
cron add sessionTarget=main \
kind="systemEvent" \
schedule.kind="cron" \
schedule.expr="0 4 * * *" \
payload.text="執行探索任務：run ~/.openclaw/workspace/scripts/run-explore-task.sh"
```

---

## 直接執行方式

手動執行：
```bash
~/.openclaw/workspace/scripts/run-explore-task.sh
```
